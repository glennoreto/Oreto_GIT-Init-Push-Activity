what is Raspberrypi

One can use Raspberry Pi to construct a typical desktop personal computer. The hardware includes Raspberry Pi, a micro SD card with an operating system installed, a constant power source, and an output display device like an old monitor or television. It is also essential to have a USB mouse and keyboard.

The purpose is, it's a computer, a real computer, that costs next to nothing, is tiny, and runs on next to no power, and needs no active cooling. You could just install Linux on and use it as a pc.
